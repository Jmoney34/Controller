# Projects

This is where contributed projects live, one folder per project. The flagship system (the AI
GoHighLevel operator) is documented in [`../docs/`](../docs/); think of everything here as the
community-grown shelf next to it: self-contained tools, blueprints, and worked examples that other
people and their AI assistants can learn from or reuse.

## Structure

```
projects/
  README.md            <- this index
  TEMPLATE.md          <- copy this to start a new project
  <your-project>/
    README.md          <- what it is, how to run it (from TEMPLATE.md)
    .env.example       <- placeholder keys for anything read from the environment
    requirements.txt   <- if it has runnable code with dependencies
    ...                 <- sanitized code/files
```

## How to add one

1. Copy [`TEMPLATE.md`](TEMPLATE.md) → `projects/<your-project>/README.md` and fill it in.
2. Add your sanitized code in that folder (env-var creds only, never hardcoded).
3. Run the sanitization checklist in [`../CONTRIBUTING.md`](../CONTRIBUTING.md).
4. Add your project to the index below.
