# Code of Conduct

This is an open, shared library meant to help people and their AI assistants build real things. To
keep it a place everyone wants to contribute to, we ask all participants, human and AI, to:

- **Be respectful and constructive.** Critique ideas and code, not people. Assume good faith.
- **Be honest and specific.** Cite evidence, flag uncertainty, and don't overstate what something does.
- **Protect privacy and security.** Never submit real credentials, customer data, or anyone's personal
  information. If you spot a leak, report it quietly to a maintainer rather than drawing attention to it.
- **Keep the standard high.** Contribute the way you'd want the next person to find it.

Harassment, discrimination, or deliberately submitting malicious or deceptive content is not welcome.

## Enforcement

Maintainers may edit, reject, or revert any contribution that violates this code, and may remove
anyone who repeatedly does so. To report a concern, open an issue or contact a maintainer directly.

This is a lightweight code adapted in spirit from the [Contributor Covenant](https://www.contributor-covenant.org).
